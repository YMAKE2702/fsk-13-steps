# fsk-13-steps

Интерактивная HTML-модель пилотного процесса ФСК:
«Пилот 13 шагов» для BPMS/Pyrus.

## О проекте

Модель визуализирует процесс от:

* договора генпроектирования,
* загрузки и проверки ПД,
* выпуска ПД в экспертизу,
* получения РнС,
* проверки РД,
* тендерного блока,
* до договора подряда.

Основа модели:

* Анализ 13 шагов
* АП v1.5
* тендерный регламент ФСК
* рабочие материалы BPMS-пилота

---

## Что реализовано в HTML-модели

### Overview

Сводная информация по пилоту:

* цели пилота
* scope
* пилотные проекты
* гейты G1 → G3
* контур систем
* ограничения пилота

### 13 Steps

Детальная модель всех 13 шагов:

* роли
* действия
* системы
* справочники
* вопросы и блокеры
* автоматические и ручные действия

### Cards

Карточный режим просмотра шагов процесса.

### Dictionaries

Справочники и сущности:

* FSK_Проекты
* FSK_РнС
* FSK_ПД
* FSK_РД
* FSK_Лот
* FSK_Контрагенты
* FSK_Договоры
* FSK_Сотрудники
* FSK_РольСотрудника
  и др.

### Graph

Визуальная схема процесса.
